# Scheduled App Controller

Scheduled App Controller is an Android application designed to help users manage their focus by blocking selected apps and notifications during specified times. Users can create customized focus modes (such as Work, Study, or Gym), select the apps they want to block, and set the start and end times for these modes.

## Features

- **Custom Focus Modes:** Create and manage custom focus modes to block specific apps and notifications.
- **Scheduled Blocking:** Set start and end times to automatically enable focus modes.
- **App Blocking:** Block selected apps to prevent distractions during focus time.
- **Notification Management:** Silence notifications from selected apps while in focus mode.
- **User-Friendly UI:** Easy-to-use interface with intuitive navigation and settings.
